# Project Context for Claude

## Common Snowflake Databases and Tables

Here are the most commonly used tables with `snow sql` CLI for data analysis:

- **coredata.customer.customers_monthly__net_revenue** - monthly net revenue by customer
- **coredata.salesforce.accounts** - Salesforce account data
- **coredata.salesforce.opportunities** - Salesforce opportunities
- **coredata.cash.cash_customer__wide** - wide table with cash customer data

## Usage

When writing queries or analyzing data, reference these tables for customer revenue, Salesforce, and cash account information.

**See `SNOWFLAKE_SCHEMA.md` for detailed table structures, column definitions, and sample queries.**

**See `reference_queries.sql` for validated query patterns from the data team.**
